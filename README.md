## kotlin
官方推荐语言能不了解吗，学习基本无成本，万一以后火了呢

**官方推荐优点：**

 - 简洁：大大减少样板代码的数量
 - 安全：避免空指针异常等整个类的错误
 - 互操作性：充分利用 JVM、Android 和浏览器的现有库
 - 工具友好：可用任何 Java IDE 或者使用命令行构建
 - 详细移驾[https://www.kotlincn.net/](https://www.kotlincn.net/ "kotlin语言中文网")
